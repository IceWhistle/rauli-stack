METHOD 3: PowerShell Native (No Python Needed)
───────────────────────────────────────────────────────────────────────────
Paste this in PowerShell on Windows PC:

Get-ChildItem C:\temp\casefiles\*.pdf | ForEach-Object { Add-Content C:\temp\analysis.txt "`n`n### $($_.Name) ###`n`n"; try { $pdf = New-Object -ComObject AcroExch.PDDoc; $pdf.Open($_.FullName); Add-Content C:\temp\analysis.txt "PDF opened successfully - manual text extraction needed"; $pdf.Close() } catch { Add-Content C:\temp\analysis.txt "Cannot auto-extract - please open in Adobe and copy text" } }; notepad C:\temp\analysis.txt
