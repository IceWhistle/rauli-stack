## MyDockFinder Setup
- **Best solution** for macOS dock on Windows: MyDockFinder ($5 on Steam)
- Uninstalled Windhawk bloat
- Steam Workshop has themes (macOS Ventura 13, Tahoe, etc.)
- Config: `C:\Program Files (x86)\Steam\steamapps\common\MyDockFinder\config.ini`
- Set `hidealltaskbar=1` to hide Windows taskbar
- Added `vsync=0` and `fps=0` for unlimited framerate (though dock apps are internally 60fps capped)

## Technical Notes
- Dock apps (MyDockFinder, Nexus, etc.) render at 60fps internally - won't match 280Hz monitor
- Windows taskbar hiding: StuckRects3 registry + TaskbarAutoHideInDesktopMode
- MyDockFinder has built-in Spotlight search (Cmd+Space or configurable hotkey)
